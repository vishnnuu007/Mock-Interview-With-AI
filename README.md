# Mock Interview with AI

An AI-powered mock interview platform that helps candidates prepare for technical and HR interviews by analyzing verbal and non-verbal communication.

## Features

* Automated mock interviews
* Voice-to-text response analysis
* AI-generated interview questions
* Confidence assessment
* Eye-contact detection
* Head-pose tracking
* Real-time feedback and evaluation

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* Bootstrap
* MySQL
* Google Gemini API

## Modules

* User Module
* Admin Module
* Interview Question Management
* Voice Analysis
* Face Detection and Recognition
* Performance Evaluation

## How to Run

1. Clone the repository.
2. Install required packages.
3. Configure the MySQL database.
4. Add your Google Gemini API key.
5. Run the Flask application.
